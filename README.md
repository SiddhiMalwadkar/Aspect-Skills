# Aspect Skills

**Aspect Skills website** is a professional training and placement-oriented institute focused on building industry-ready skills for students and working professionals.  
This repository contains the source code for the official **Aspect Skills website**, designed with a modern, responsive, and corporate-friendly UI.

---

## Overview

The website provides detailed information about courses, services, placements, and corporate training programs. It is built using frontend technologies only and follows clean UI/UX and professional design standards.

---

## Features

- Responsive and modern UI using **HTML5, CSS3, and Bootstrap**
- Mega dropdown navigation for easy course browsing
- Placement overview and trusted client showcase
- Enquiry form integrated with **FormSubmit** (no backend required)
- Smooth animations using **Animate.css** and **AOS**
- Clean and professional layout suitable for corporate branding

## Enquiry Form

The website includes an enquiry form that sends user submissions directly to email using **FormSubmit**, eliminating the need for any backend server or database.
